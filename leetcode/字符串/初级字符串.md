
### 344.反转字符串

- 双指针法：头尾指针进行掉换，时间复杂度为O(N)

```
class Solution {
public:
    void reverseString(vector<char>& s) {
        int start=0;
        int end=s.size()-1;
        while(start<end){
            int tmp=s[start];
            s[start]=s[end];
            s[end]=tmp;
            start++;
            end--;
        }

    }
};
```
